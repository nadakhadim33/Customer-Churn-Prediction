# Customer Churn Prediction

##  Overview

This project predicts whether a customer will leave a telecom company using machine learning.

---

##  Technologies Used

* Python
* Pandas
* Scikit-learn

---

##  Data Processing

* Cleaned missing values in TotalCharges
* Converted categorical data using One-Hot Encoding
* Removed unnecessary columns

---

##  Model

* Random Forest Classifier
* Used class balancing to improve performance

---

##  Results

* Accuracy: 78%
* Recall (Churn=1): 73%
* F1-score: 63%

---

##  Key Insights

* Customers with high total charges are more likely to leave
* Short-term customers have higher churn risk
* Monthly charges significantly impact churn

---

##  What I Learned

* Handling imbalanced data
* Improving model performance using class_weight
* Extracting business insights from data
